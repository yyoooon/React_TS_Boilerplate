# React_TS_Boilerplate

## 0. 실행 방법
- 터미널에서 `yarn install`을 입력해 패키지를 설치합니다.
- 이후에 `yarn start`를 입력해 app을 실행시킵니다.

</br>

## 1. 기술 스택

- UI라이브러리: `React` (함수형 & hook)
- 언어: `Ts`
- 스타일: `Emotion`
- 서버통신: `Axios`
- 라우팅: `react-router-dom`
- 번들링: `Webpack5`
- 트랜스파일: `Babel`
- 포맷팅&린팅: `EsLint`, `Prettier`

</br>
